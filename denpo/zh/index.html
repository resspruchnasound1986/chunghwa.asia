Denpo

用 Markdown 寫 Telegraph 文章

贊助 Denpo


綜覽

Denpo 是一個將本地 Markdown 文檔發佈到 Telegraph 的軟體。Telegraph 不識 Markdown 也對一些ＨＴＭＬ元素支援不完全（例如 <code>。無奈 Telegraph 無法顯示 <h1> 與 <h2>）。此外，Denpo 亦提供類似於 TinyURL 這樣的短鏈接服務，以及，通過 Cloudflare Workers 的重定向（需要您手動搭建，如何？）。其後能夠跳脫來自 ISP、學校和政府的內容審查。無料。


軟體下載

  • （二〇二〇年四月十日，beta 0.2 build 1）要求 macOS 10.15 或更高的系統版本。點擊此處以下載。

Markdown 支援

Denpo 支援：

  • 作為標題的 <h1>
  • 基礎：<h3>、<h4>、<em>、<strong>、<code>
  • 鏈接：<a>
  • 引用：<blockquote>

將不會支援：

  • <h1>、<h2>
  • 清單
  • 表單
